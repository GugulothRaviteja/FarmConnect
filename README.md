# FarmConnect
Farmer to Market supply Chain Project using Core Java and Exception Handling
A simple Java console project where:
- Farmers can add, view, and check sold-out products.
- Buyers can view and purchase products.
- Uses arrays and custom exception handling.

### Features
- Farmer & Buyer menus
- Add/view products
- Order tracking
- Error handling (OutOfStock, ProductNotFound)

### Technologies
- Core Java
- Arrays
- Exception Handling
